This file contains extensions of the model-checker software developed by Benjamin Brast-McKie, of which this repository is a fork. So far, the following revisions have been incorporated: 
--an extension of the bimodal semantics to include metric tense operators, including Next, Previous, Since, and Until. 
--newly defined operators meant to capture progressive tenses. Right now, these include operators meant to capture the present progressive (PreProg) and present perfect (Pastprog). 

In development: 
--a semantics for Linear Temporal Logic 
--more progressive tense operators
